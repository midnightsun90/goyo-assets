# 고요 개인정보처리방침

시행일: 2026-07-06

'고요'(이하 "앱")는 사용자의 개인정보를 소중히 여기며, 처음부터 개인정보를 모으지 않는 방향으로 설계되었습니다.

## 1. 수집하는 정보

**계정이 없습니다.** 이름, 이메일, 전화번호 등 개인을 식별할 수 있는 정보를 수집하지 않습니다.

**수행 기록은 기기에만 저장됩니다.** 읽은 구절, 완독 진도, 연속 수행 일수, 직접 남긴 한 줄 기록(되새김)은 모두 사용자의 기기 안에만 저장되며 외부로 전송되지 않습니다. 종교적 신념에 관한 어떠한 정보도 수집하지 않습니다.

**익명 사용 통계(선택).** 앱을 더 낫게 만들기 위해 익명 통계 도구(Aptabase)로 다음을 수집할 수 있습니다: 앱 실행 여부, 화면 사용 이벤트(예: 과 완료), 기기 모델과 운영체제 버전. 이 정보에는 사용자를 식별할 수 있는 값이 포함되지 않으며, 광고에 사용되지 않고, 제3자에게 판매·제공되지 않습니다. 앱의 설정에서 언제든 끌 수 있습니다.

## 2. 알림

매일 알림은 사용자가 정한 시각에 기기 안에서 울리는 로컬 알림입니다. 알림을 위해 어떤 정보도 서버로 전송되지 않습니다.

## 3. 데이터 보관과 삭제

모든 수행 기록은 기기에 저장되므로, 앱을 삭제하면 함께 삭제됩니다. 별도의 서버 보관본이 없습니다.

## 4. 제3자 제공

개인정보를 제3자에게 제공하지 않습니다.
